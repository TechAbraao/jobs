jobs-cli

Exemplos de uso do comando search (CLI para buscar vagas na Gupy direto do terminal).

Opções disponíveis: --city, --limit, --keyword, --type · veja a tabela completa no fim da página.

Busca básica

Busca padrão

jobs-cli search

Trazer mais resultados

jobs-cli search --limit=30

Trazer só o essencial

jobs-cli search --limit=3

Por cidade

Rio de Janeiro

jobs-cli search --city="Rio de Janeiro"

Belo Horizonte

jobs-cli search --city="Belo Horizonte"

Recife

jobs-cli search --city="Recife"

Porto Alegre, com mais resultados

jobs-cli search --city="Porto Alegre" --limit=20

Por tipo de vaga

Estágio

jobs-cli search --type="Estágio"

Jovem Aprendiz

jobs-cli search --type="Jovem Aprendiz"

Efetivo

jobs-cli search --type="Efetivo"

Por palavra-chave

Vagas de dados

jobs-cli search --keyword="Dados"

Vagas de suporte

jobs-cli search --keyword="Suporte"

Vagas de design

jobs-cli search --keyword="Design"

Cargo específico

jobs-cli search --keyword="Desenvolvedor Python"

Combinando filtros

Estágio em tecnologia no Rio de Janeiro

jobs-cli search --city="Rio de Janeiro" --type="Estágio" --keyword="Tecnologia"

Jovem aprendiz em São Paulo, poucos resultados

jobs-cli search --type="Jovem Aprendiz" --limit=5

Vagas de dados efetivas em Belo Horizonte

jobs-cli search --city="Belo Horizonte" --keyword="Dados" --type="Efetivo" --limit=15

Erros comuns

O --type valida o valor antes de rodar a busca.

Tipo inválido

jobs-cli search --type="CLT"
Error: Invalid value for '--type': 'CLT' is not one of 'Efetivo', 'Estágio', 'Estagiário', 'Jovem Aprendiz'.

Cidade com espaço, sem aspas

jobs-cli search --city=São Paulo
Error: Got unexpected extra argument (Paulo)

Sempre envolva nomes com espaço em aspas: --city="São Paulo".

Referência de opções

OpçãoTipoPadrãoDescrição
--citystrSão Paulo Filtra vagas por cidade.
--limitint10 Quantidade de resultados retornados.
--keywordstrN/A Palavra-chave para buscar no título e na descrição das vagas.
--typestrEfetivo Tipo de vaga: Efetivo, Estágio / Estagiário, Jovem Aprendiz.
