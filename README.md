## 1. Sobre
### 1.1. InformaГ§Гөes Gerais
- CLI em Python para buscar vagas de emprego diretamente pelo terminal, consumindo a API da Gupy.
- Projeto de cunho acadГӘmico e open-source, com o objetivo de ajudar a comunidade a filtrar e encontrar oportunidades de emprego com mais facilidade.

### 1.2. Tecnologias
- Python
- Httpx
- Typer
- Rich
- Pytest

### 1.3. DocumentaГ§ГЈo
#### 1.3.1. Comando `search`

- Busca vagas de emprego na plataforma Gupy com filtros de cidade, tipo de contrato e palavra-chave.

```bash
jobs-cli search [OPTIONS]
```
##### 1.3.1.1. Tabela

| OpГ§ГЈo | Atalho | Tipo | PadrГЈo | DescriГ§ГЈo |
|---|---|---|---|---|
| `--city` | `-c` | `str` | `None` | Filtra vagas por cidade. |
| `--limit` | `-l` | `int` | `10` | Quantidade de resultados retornados. |
| `--keyword` | `-k` | `str` | `None` | Palavra-chave para buscar no tГӯtulo e na descriГ§ГЈo das vagas. |
| `--type` | `-t` | `str` | `Efetivo` | Tipo de vaga a ser filtrada. OpГ§Гөes: `Efetivo`, `EstГЎgio`, `Jovem Aprendiz`. |
| `--state` | `-s` | `str` | `None` | Filtra vagas por estado. |
| `--output` | `-o` | `str` | `None` | Nome do arquivo `.txt` para salvar os resultados (salvo em `jobs/data/`). |
| `--enterprise` | `-e` | `str` | `None` | Filtra vagas por empresa. |
- Busca padrГЈo

```bash
$ jobs-cli search
```

```bash

$ pip install -e .
```
